Coding Sketch #1

Art Imitation: Find a painting or picture by a famous artist. Create an interpretation of it using 2D primitives in P5.JS. You can to be accurate, or create an interesting abstraction based on the image. Look at the 2D primitives on https://p5js.org/reference/.

Example by Joe McKay

Make sure to look at the P5 reference. Bonus for using more advanced 2D primitives -- quads, arcs, and I encourage you to lookup how to use Vertexes!
