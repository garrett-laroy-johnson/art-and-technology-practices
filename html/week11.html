Project 2

Using p5*js, use the canvas to create an interactive artwork. It may be a short comic or animation, cubistic and abstract. It my be inspired by some of the works we've seen before, or completely your own.

Required Elements

The following elements *must* be in your piece

  • a moving element (bounce off walls, circular, spiral, or sine wave movement)
  • a function that *you* have made to organize your code
  • use rotate() or scale() to transform a drawn element (circle, square, rectangle, polygon, line, etc.) at least once in your code.
Tips

Get all of the required coding elements in your sketch. These are worth 1/3 of your project grade! Start by putting these elements in the same sketch (perhaps before endeavoring to bring together the art work).

For this assignment, you should not need to borrow and cite any code (it is easy for me to see if you submit code which is beyond your ability!).

These coding techniques have been taught in class. If you do not remember or know how to do something -- use your resources to brush up. It is normal not to remember every single thing -- make it easy on yourself and use your resources!
