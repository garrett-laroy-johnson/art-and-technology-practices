Coding Sketch #7: Coding Sketch

Create a p5 sketch in which you compose *your own* object to animate a shape. Your object should:

  1. accept a color parameter to assign as one of its properties (e.g. this.color)
  2. define and use 2 methods: one to move, and one to show
  3. Create 4 instances of these objects with four different colors. Call their move and show methods inside the draw loop.
